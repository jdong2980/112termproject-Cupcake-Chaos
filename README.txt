Introduction:
Cupcake Chaos is similar to the game series Overcooked and Papa’s Cupcakeria. This is a single-player game that requires the user to build multiple orders of cupcakes in the kitchen over a 5 day time span by moving around the “kitchen” which contains all of the toppings, a sink, an oven, cupcake cup holders, frosting, etc to be collected.  After each serving, the user will get a rating out of 5 stars as well as a tip based on how fast and accurately they completed the order. The money obtained can be used in the game “store,” allowing the user to better maximize the amount of orders they complete. The objective of the game is to complete as many orders as possible, with the game ending when the user has no money left. 

Installation Instructions: 
In order to run the project, the following modules must be first be downloaded:
cmu_112_graphics from: https://www.cs.cmu.edu/~112/notes/hw7.html 

Run Instructions:
Run ‘main.py’

List of Shortcuts for Game:
[space] to pick up and drop items onto the plate
[enter] to submit order
[up], [down], [right], [left] arrows to move sprite around board 
[p] to pause game

Fonts:
No additional fonts need to be installed.
